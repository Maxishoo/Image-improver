# Image Improver - Улучшение фото с помощью нейросети

Простое веб-приложение для автоматического улучшения тёмных и некачественных фотографий. Работает прямо в браузере с помощью нейросети Zero-DCE++.

## Что умеет

- **Делает тёмные фото ярче** - нейросеть автоматически улучшает освещение
- **Сравнение до/после** - удобный слайдер для сравнения оригинала и результата
- **Быстрая работа** - обработка происходит в браузере, ничего не загружается на сервер
- **Скачивание результата** - можно сохранить улучшенное фото
- **Работает везде** - открывай на компьютере или телефоне

## Что нужно для запуска

- Node.js 18 или новее
- Docker (не обязательно, но удобно)
- Любой современный браузер (Chrome, Firefox, Edge)

## Установка

### Обычный запуск

1. **Скачай проект:**
```bash
git clone <ссылка-на-репозиторий>
cd image-improver
```

2. **Установи зависимости:**
```bash
npm install
```

3. **Запусти:**
```bash
npm run dev
```

4. **Открой в браузере:**
```
http://localhost:5173
```

### Через Docker

Если не хочешь возиться с установкой Node.js:

```bash
docker-compose up --build
```

И открывай `http://localhost:5173`

## Как пользоваться

1. **Загрузи фото** - нажми "Выберите файл" и выбери тёмное или некачественное изображение
2. **Подожди** - нейросеть обработает фото (обычно 2-10 секунд)
3. **Сравни** - подвигай слайдер влево-вправо, чтобы увидеть разницу
4. **Скачай** - нажми кнопку "Скачать", чтобы сохранить результат

## Как это работает

1. После выбора фотографии, она загружается в сервис
2. Фотография сжимается до небольшого размера (чтобы быстрее проходила обработка)
3. Фотография отправляется в Web Worker (чтобы не тормозило интерфейс)
4. Нейросеть Zero-DCE++ анализирует фото и возвращает кривые улучшения
5. К исходной фотографии применяются кривые
5. Возвращается результат и отрисовывается обработанное изображение
6. Появляется возможность скачать улучшенное фото

## Технологии

- **TypeScript** - пишу на нём, чтобы было меньше ошибок
- **Vite** - быстрая сборка проекта
- **ONNX Runtime** - запуск нейросетей в браузере
- **Web Workers** - обработка в фоне
- **Docker** - чтобы работало у всех одинаково
- **Zero DCE** - нейросеть для обработки фотографий
